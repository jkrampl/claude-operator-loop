# One-pager / product sheet template

## Target
Single printable page (A4 or letter). Partner leave-behind, trade fair handout, chain buyer sheet, press kit insert. Content-only (HTML/PDF formatting is project's print team).

## Structure

1. **Header strip** (top 15% of page)
   - Brand logo position (left typically)
   - Product name + variant (headline-size)
   - Category / type / vintage if applicable

2. **Hero line** (1 sentence, under the header)
   - Sets the positioning in one line
   - Example: "Flagship late-harvest Frankovka from the Little Carpathians"

3. **Key value props** (3-5 bullets)
   - Each bullet: 5-10 words, concrete
   - Mix of quality proof + practical details
   - Example:
     - Gold medal, Vinalies Internationales 2026
     - Hand-picked single vineyard
     - 750ml bottle, 13.5% ABV
     - Stocked in 200+ SK outlets
     - Price tier: premium

4. **Product details table** (structured data)
   - Varietal(s)
   - Vintage
   - Region / appellation
   - Alcohol %
   - Bottle size(s) available
   - Minimum order quantity (for B2B)
   - Case / pallet configuration
   - EAN / SKU codes
   - Shelf life / best-by guidance

5. **Story paragraph** (80-150 words)
   - The "why this wine" narrative
   - Specific enough to differentiate; short enough to scan
   - No flowery tourism-brochure prose

6. **Proof block**
   - Medals / awards list (full format: competition + year + level, max 5 most recent/prestigious)
   - One quote from press / critic if available (with attribution)

7. **Footer strip**
   - Company full legal name
   - Contact: single email + single phone + single URL
   - Address (registered + tasting room if different)
   - Legal identifiers (IČO / VAT if B2B sheet)

## Format notes

- **Pagination discipline.** If content overflows, compress the story paragraph or trim medal list — never spill to a second page.
- **Scannable first, readable second.** Reader's eye should take in the key info in 10 seconds.
- **Version dates.** Footer-corner: "v2026-04" or similar — one-pagers go stale quickly.

## Bilingual (if applicable)

Usually separate sheets per language, not mixed. Same structure + same data, parallel phrasing.

## Quality checklist

- Does a reader get the core value in 10 seconds?
- Every factual claim accurate + current?
- Medals cited in full format (competition + year + level)?
- No AI-isms, em-dashes, promotional filler?
- Brand rules respected?
- Would this printed at A4 look dense but readable, not cramped?
- Contact info is ONE point of each (not five emails)?

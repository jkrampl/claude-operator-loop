# Social caption template

## Target
Single post or post series (thread). Platform-specific length and tone.

## Platform-specific rules

### Facebook
- **Max per post:** no hard limit, but first 2 lines are what shows before "See more"
- **Bilingual (SK+EN):** native multilingual post — both languages in one post, separated by a visual break (line break + separator or emoji)
- **Tone:** conversational, story-forward
- **Character count:** typically 80-250 for engagement

### Instagram
- **Max caption:** 2,200 characters
- **Bilingual:** SK + EN in one caption, visually separated (e.g. line break + "🇬🇧 ————— 🇬🇧" or similar)
- **Tone:** visual-first, caption supports the image
- **First line matters most** (shows in feed before "more")
- **No hashtags** per project convention (CLAUDE.md rule for ACME)

### X / Twitter
- **Single post:** 280 characters
- **Thread:** 2-8 posts typical, each under 280
- **Language:**
  - SK-only for operational posts (daily ops, store hours, minor news)
  - SK + EN thread for campaigns (SK thread first, EN thread second)
- **Tone:** punchy, direct, often leads with a hook
- **First tweet** should work as a standalone in feed

### LinkedIn (if applicable)
- **Max post:** 3,000 characters
- **First 3 lines** are the preview
- **Tone:** professional but human; avoid corporate speak
- **Format:** short paragraphs, often with line breaks between each sentence for scannability

### Stories / Reels
- **One language per story** (not bilingual side-by-side)
- **Caption on Reels:** SK + EN + enable auto-translation if available
- **On-screen text:** short, high-contrast, timed to visual

## Structure (per post)

1. **Hook** — first line earns the "more" click. Specific, intriguing, or bold.
2. **Context** — what's the news / value / story? 1-3 sentences.
3. **Details** — specifics, numbers, names.
4. **CTA or signoff** — what should the reader do? (Visit shop, book tasting, DM us, read more at...)

## Bilingual separation options

- `---` (three dashes)
- `🇬🇧 🇬🇧 🇬🇧`
- `/ / /`
- Double line break + language flag emoji

Pick one per project convention and stay consistent across posts.

## Quality checklist

- First line stops the scroll?
- Language matches platform convention for bilingual?
- No hashtags per project convention?
- No AI-isms, em-dashes, filler transitions?
- Does it respect the medal-communication framework if announcing an award? (For ACME: competition + year + wine + medal level required; no inflated generics)
- Brand rules respected (e.g. ACME never "family winery"; never mention hectares)?
- Character counts within platform limits?
